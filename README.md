# 專案名稱 | Netflix 選片小幫手
# 組員 | 黃建廷、吳秉宸
# 專案簡介 | 主要是把幾部電影的資料放在 script.js 裡，由 JavaScript 動態產生卡片並顯示在網頁上
# 功能特色：

* **導覽與版面**

  * 導覽列分為「首頁／熱門電影／推薦表單／我的收藏」等區塊，便於切換與瀏覽。 
  * 首屏標語「探索你的下一部好電影」與行動按鈕「開始使用」，引導快速上手。 

* **熱門電影**

  * 提供「本週熱門作品」區塊，作為使用者進站的第一層選片參考。

* **個人化推薦表單**

  * 表單欄位包含：姓名（至少 2 個字元）、Email（格式驗證）、年齡（13–100）、偏好類型（多選）與其他偏好說明（提供剩餘字數提示），並需勾選同意條款後才能送出。
  * 送出後顯示「推薦成功」訊息與推薦清單區塊。 

* **我的收藏**

  * 提供「我的收藏電影」頁面與「清空收藏」操作，未收藏時顯示提示文字。
 
# 技術亮點
- HTML5 語意化結構與 RWD 版面。
- CSS3 + Bootstrap 5 風格與排版。
- JavaScript 實作互動（表單驗證、收藏區塊顯示、導覽切換等）。
- DOM 操作（查詢節點、監聽事件、動態插入內容）。
- HTML5 Constraint Validation API（最小字數、年齡範圍、Email 格式、條款勾選等）。
